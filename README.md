# AuthVault - Secure Authentication System

AuthVault is a professional Login & Signup Authentication System developed using HTML, CSS, JavaScript, and PHP Sessions (without SQL/MySQL).

## Features
- User Signup System
- Secure Login Authentication
- PHP Session Management
- Protected Dashboard
- Logout Functionality
- JavaScript Form Validation
- Password Show/Hide Feature
- Professional Dark UI Design
- Responsive Layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP

## Project Structure
AuthVault/
│── index.php
│── signup.php
│── dashboard.php
│── logout.php
│── style.css
│── script.js

## How to Run
1. Install XAMPP
2. Place project folder inside `htdocs`
3. Start Apache server
4. Open browser and run:

http://localhost/AuthVault

## Purpose
This project was developed to understand frontend and backend integration, authentication systems, session management, and professional UI design in web development.
